Acoustic-optical signal classification

https://github.com/omrigo5/Signal-Classification

Acoustic-optical time-series signal classification using Pytorch - LSTM and 1D CNN with a self-collected dataset. Such a model could be useful for recording a meeting which takes place in an isolated building for example. One can record the vibrations of the windows (or any other nearby object) and reconstruct the audio signal.

Hotel reservation cancellation prediction using Scikit-Learn. Tabular data was imported from Kaggle datasets and analyzed using the Pandas library.

Wheat-head detection

https://github.com/omrigo5/Wheat-Detection

Object detection of wheat-heads using a Torchvision Faster-RCNN pretrained model. The dataset was imported from a Kaggle competition and from an additional set of augmented images
